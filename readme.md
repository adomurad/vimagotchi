# Vimagtochi

## Instalation

```lua
{
  "adomurad/vimagotchi",
  config = function()
    local vimagotchi = require 'vimagotchi'
    vimagotchi.setup {}

    vimagotchi.open()
  end
}
```

## Options

## Commands

### Open the Vimagotchi window

```
:Vimagotchi Open
```

### Close the Vimagotchi window

```
:Vimagotchi Close
```
